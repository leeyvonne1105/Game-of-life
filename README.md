* 目的: 表現出至少有 3 種生命型態：靜止的、原地動作的、會移動的
* 作品畫面: 視窗左上方有一個glider gun不斷發射子彈，視窗的左下方和右上方分別有兩個脈衝星，他們運行到最後會靜止不動，或是呈現動態平衡。
* 額外功能: glider gun的子彈是固定一個時間間隔發射的，若想要改變發射時間可以按"g"延遲發射
